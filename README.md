# ThreadPool

Реализация паттерна ThreadPool. Паттерн ThreadPool представляет собой класс, который изначально создаёт определённое количество потоков и хранит их в очереди, вместо того, чтобы каждый раз создавать новый поток, что является достаточно ресурсоёмкой операцией. Также такой класс обладает возможностью добавлять и хранить задачи, а также распределять их по потокам.

---

## Архитектура

Проект имеет следующую архитектуру:

- **Класс ThreadPool**
- **Структура Task**

### Функции класса:
- `addTask` – для добавления задач
- `addTaskWithResult` – для добавления задач с возвращением результата
- `getThreadPoolInfo` – получение информации о потоках, их состоянии, выполняемых ими задачах
- `shutdown` – для завершения работы ThreadPool вручную

### Тестовые функции:
- `sleepTask` – имитация длительной работы
- `sumRandomNumbers` – вычисление суммы случайных чисел
- `sumVectors` – объединение строк

Консольный интерфейс
